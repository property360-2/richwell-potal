## 5. Usage Notes

*   **Online Enrollment**: You can test the online enrollment flow at the public endpoint. It will automatically assign the latest active curriculum for the selected program.
*   **Logout**: Always log out before switching between different roles (e.g., switching from Student to Cashier) to ensure session tokens are refreshed.
*   **Data Reset**: To restore these accounts to their original state, run:
    ```bash
    python manage.py seed_full_v2 --wipe
    ```
